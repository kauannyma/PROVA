o que é clone ?

É cópia de um repositório remoto para o local.

o que é um commit ?
É o empacotamento de alterações.

O que é um repositório?

É pasta de projetos onde os arquivos são guardadas.

O que é merge ?

É a juncão de branch.

O que é pull?

É puxar informações do repositório remoto para o local.

O que é branch?

São as ramificações do repositório. 

O que é pull resquest?

É a solicitação de alteração.

O que é fork ?

É a cópia de um repositório de terceiros.

O que é um vcs? 

É um software de controle de versão.
